# Что изменено в этой учебной версии

- Проект собран как полноценная реализация TeamFinder для варианта 3.
- Настройки читаются из `.env`; `TASK_VERSION` реально выбирает папку шаблонов.
- Добавлен режим `DB_ENGINE=sqlite` для быстрых локальных тестов и `postgresql` для основной разработки.
- Усилена валидация GitHub-ссылок: проверяется настоящий домен GitHub.
- Телефон нормализуется к формату `+7XXXXXXXXXX` и проверяется на уникальность.
- `Skill.name` сделан уникальным и индексируемым.
- Добавлены индексы для списка проектов и фильтрации.
- Добавлены базовые автотесты для регистрации, проектов, участия и навыков.
- Добавлена команда `python manage.py seed_demo` для тестовых пользователей и проектов.
